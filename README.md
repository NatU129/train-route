# Simple Train Routes

### Requirements
node@12.16.0


### How to run project
1. Install packages for both client and server
```bash
cd client/train-route
npm install

cd ../../server
npm install
```
2. Build client
```bash
npm run build-client 
```
3. Run server
```bash
npm run build-server 
npm run start-server ./src/route.csv
```
